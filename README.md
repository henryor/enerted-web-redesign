## 🚀 Quick Start

### Para Desarrolladores

```bash
# 1. Clonar repositorio
git clone https://github.com/henryor/enerted-web-redesign.git
cd enerted-web-redesign

# 2. Instalar dependencias
npm install

# 3. Configurar variables de entorno
cp .env.example .env.local
# Editar .env.local con tus valores

# 4. Ejecutar en desarrollo
npm run dev

# 5. Acceder a http://localhost:3000
```

## 💡 Oportunidades Identificadas

### 1. SEO - Falta de Contenido
- 0 artículos blog (pierde tráfico long-tail)
- 25-30 keywords sin ranking que podría capturar
- Autoridad de dominio 28 (meta: 40+)

### 2. SEM - Sin Campañas Activas
- Google Ads: $0/mes gastado
- Potencial: $7,000/mes → 70-80 leads
- Costo por lead reducible de $400 → $150

### 3. Conversión - Sin Automatización
- Links WhatsApp manuales
- Sin GA4 (0% rastreo de conversiones)
- UX sin optimizar para móvil

### 4. Design - Sitio Anticuado
- WordPress tradicional
- Velocidad 3.5s (debe ser <1.5s)
- Diseño no responsivo 100%
