# Código del Sitio Web ENERTED
## Estructura
- web-src/: Componentes y librerías React
- web-app/: Layout y páginas Next.js
- web-public/: Archivos públicos (imágenes, etc)

## Para ejecutar localmente

\`\`\`bash
cd enerted-web
npm install
npm run dev
\`\`\`

El sitio estará en http://localhost:3000

## Componentes Creados (Día 1)

- Header.tsx: Navegación + logo
- Hero.tsx: Sección hero fullscreen
- ServiceCard.tsx: Tarjeta de servicio individual
- ServiceGrid.tsx: Grid de servicios (3 columnas desktop)
- Footer.tsx: Footer con links
- WhatsAppButton.tsx: Botón flotante WhatsApp
- lib/constants.ts: Datos de servicios
